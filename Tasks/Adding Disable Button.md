```abap
TABLES: SFLIGHT, SSCRFIELDS.

DATA: GV_INPUT TYPE SCREEN-INPUT VALUE 1.
CONSTANTS: CV_DISP TYPE TEXT40 VALUE 'Display',
           CV_CHAN TYPE TEXT40 VALUE 'Change'.

PARAMETERS: PA_CAR TYPE SFLIGHT-CARRID MODIF ID GR1.
SELECT-OPTIONS SO_CON FOR SFLIGHT-CONNID MODIF ID GR1.

SELECTION-SCREEN BEGIN OF LINE.

SELECTION-SCREEN PUSHBUTTON 1(10) BTN USER-COMMAND VIEW.

SELECTION-SCREEN END OF LINE.

INITIALIZATION.

  PA_CAR = 'AA'.
  BTN = CV_DISP.

AT SELECTION-SCREEN.

  IF SSCRFIELDS-UCOMM = 'VIEW'.

    IF BTN = CV_DISP.

      GV_INPUT = 0.
      BTN = CV_CHAN.

    ELSEIF BTN = CV_CHAN.

      GV_INPUT = 1.
      BTN = CV_DISP.

    ENDIF.

  ENDIF.

AT SELECTION-SCREEN OUTPUT.
  LOOP AT SCREEN.

    IF SCREEN-GROUP1 = 'GR1'.

      IF GV_INPUT = 0.

        SCREEN-INPUT = 0.

      ELSEIF GV_INPUT = 1.

        SCREEN-INPUT = 1.

      ENDIF.

    ENDIF.

    MODIFY SCREEN.

  ENDLOOP.
```
